# apps/web

Astro + TypeScript (strict) frontend for exitMediaGroup.

## Prerequisites

- Bun `>= 1.3`

## Commands

- `bun install` - install dependencies
- `bun run dev` - start local dev server
- `bun run build` - build production output
- `bun run preview` - preview production output
- `bun run check` - run Astro type/content checks
- `bun run content:validate` - validate locale content JSON against schemas
- `bun run lint` - run ESLint
- `bun run lint:fix` - auto-fix lint issues
- `bun run format:check` - verify formatting
- `bun run format` - format files with Prettier
