# Skills Strategy for This Project

## Skill Used Now
- `find-skills`: used to discover additional installable skills for animation/frontend workflows.

## Current Session Skills and When to Use Them
- `find-skills`: when you want to discover installable skills for a specific task.
- `security-best-practices`: run for explicit security reviews (JS/TS portions).
- `skill-creator`: use when creating custom internal skills for repeated workflows.
- `skill-installer`: use to install curated or repo-based skills.

## Practical Workflow
1. Start with `docs/project-plan.md` task order.
2. Before each phase, run `find-skills` for targeted enhancements (example: `astro performance`, `gsap scroll`, `i18n content`).
3. If no good skill is available, continue with project rules in `AGENTS.md`.
4. After MVP, run `security-best-practices` on frontend code and data handling.

## Note
- If the skills registry is unreachable in your environment, proceed with local project rules and prompt pack; do not block delivery.
